Dino Wars



overview

Minigame focusing on multiplayer and physics interactions. Made in Unity c# using the Photon networking package.

my role

I was the primary programmer for this project. I wrote around 800 lines of Unity code and implemented the following features:

  • networking via Photon
  • cannon movement
  • firing logic
  • ammo/block purchase system
  • networked physics interactions
  • block handling (pick up, rotate, scroll
  • different block/ammo properties
  • game logic (victory, defeat, setup, etc.)

Dino wars is a 3d Angry Birds clone using multiplayer: where one player builds a tower out of limited resources, and the other tries to knock it down by firing dinosaurs at it.
